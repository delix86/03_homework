1.	- скачать проект
	- создать папку "build" в корне проекта и перейти в неё
	- помощью cmake собрать и сбилдить
	- скопировать файл "high_scores.txt" в папку "build"
	- из текущей папки "build" запустить .exe из папки \Debug как в примере:
		F:\Programming\Projects\OTUS\CPP\HomeWork\03_homework\build> .\Debug\guess_the_number.exe -max 555

2. Ключ -table - выводит таблицу результатов.

3. Ключ -max [max_value] - задать максимальное число для игры max_value.

4. Ключ -level [level] - задать уровень сложности игры (1 - 10, 2 - 50, 3 - 100)

5. Без ключа, по дефолту максимальнае занчение задано 100
